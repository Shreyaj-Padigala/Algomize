# Algomize
Trading Bot for BTC/USDT


The AI Agent-Based Crypto Trading Bot is a web application that autonomously analyzes BTC/USDT price data from the BloFin exchange and executes trades on behalf of users according to configurable trading strategies.
The system utilizes multiple specialized analytical agents to evaluate price action, market structure, RSI indicators, and confluence factors across multiple timeframes (15-minute and 1-hour charts). These agents collaborate through a centralized orchestrator and final decision engine to determine optimal trade entries and exits.


The platform provides users with a dashboard displaying active agents, current and lifetime performance metrics, and live chart data derived directly from BloFin market feeds.
